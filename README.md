Скрипт создает report.txt в той же папке, где находится сам.
Чтобы прекратить редактирование текущей даты, введите "стоп" в строке "Суть обращения"
Чтобы закончить редактирование отчета и сформировать докумет, введите "стоп" в графе "Дата"
Если оставить поле "Суть обращения" пустым путём нажатия "Enter", скрипт заполнит выбранную дату как "Не было звонков"
